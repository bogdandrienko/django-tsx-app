PyQt6(ui) + aiohttp(web) + sqlite3
- автозагрузка - перейдя к папке с файлом, нажмите клавишу с логотипом Windows + R, напечатайте shell:startup, затем нажмите ОК. Откроется папка
Автозагрузка. Скопируйте и вставьте ярлык приложения из папки с файлом в папку Автозагрузка.

- старт в трее, после задержки в 30 секунд
- при закрытии сворачивается в трей
- логи в файлы каждый час, дополнять файл

worker_saver(loop):
- берёт из api/db -> Message(id: int, date_time: datetime, message: str) -> sqlite3

worker_sender(loop):
- берёт из sqlite3 -> JSON -> aiohttp(fastapi/fasthttp)
- в случае успешного ответа от сервер, удаляет эту запись по id
